# 📧 Phishing & Network Traffic Analysis

## 📌 Scenario
Simulated phishing attacks and network traffic analysis to identify malicious activities and indicators of compromise.

## 🎯 Objective
- Analyze phishing emails
- Extract and validate IOCs
- Detect malicious network traffic

## 🛠️ Tools Used
- Wireshark
- VirusTotal
- AbuseIPDB
- MXToolbox

## 📧 Phishing Analysis
- Investigated 15+ phishing emails
- Analyzed:
  - Email headers
  - Embedded links
  - Attachments

## 🔎 IOC Extraction
- Extracted:
  - Malicious domains
  - Suspicious IP addresses
  - URLs and file hashes

- Validated using:
  - VirusTotal
  - AbuseIPDB

## 🌐 Network Traffic Analysis
- Captured and analyzed packets using Wireshark
- Identified:
  - Command & Control (C2) communication
  - Port scanning activity
  - Abnormal traffic patterns

## 📊 Findings
- Detected phishing indicators in email headers
- Identified malicious communication channels
- Confirmed suspicious IP reputation

## ⚠️ Conclusion
- Multiple phishing attempts classified as **True Positives**
- Malicious traffic patterns detected and analyzed

## 📚 Key Learnings
- Strong understanding of phishing attack patterns
- Practical experience with packet analysis
- Improved IOC identification and validation skills
